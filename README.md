# ultrashman

## 概要

- 情報工学ゼミ2024の作品，2025年のオープンキャンパスにて展示した作品。
- 研究室:安藤研究室
- ゴミを種類を自動判別して捨ててくれるシステム。

## コードの説明
1. Background Subtraction.py
  - 背景差分プログラム
  - 現在の画像と少し前の画像を比較することで，カメラ画像中のどこが変化したか読み取る。
  - ゴミが台に載ったことを認識するために利用した。
  - 今後タッチセンサ等に置き換わるときには必要ない。

2. Color recognition.py
  - 色認識プログラム
  - 画像中の 〇 色の面積が一番広い所を探す。
  - ゴミ箱の位置を読み取るために利用した。
  - 誤作動しないないよう，色認識しない場所にはマスク処理。
  - 認識可能な色は，赤・青・黄の３種類。

  - # 動作ex.(赤色を認識するパターン)
    - RGB -> HSV変換。
    - 各画像処理を行い赤色を抽出。
    - 一番赤色の面積が広い所を計算。
    - その面積の重点を返す。
